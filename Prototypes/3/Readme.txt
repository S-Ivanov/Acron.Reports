Прототип 3.0

Демонстрируется работа Java-клиента с REST-сервисом с использованием базовой аутентификации http.

Java-клиент устанавливает имя пользователя и пароль, затем запрашивает REST-сервис, сохраняет возвращаемый сервисом байтовый поток в виде файла и открывает его.

Идентификация пользователя определена в файле конфигурации сервиса:
user = testuser
password = testpassword
